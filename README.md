## 简介

**风灵月影宗** 是一款专为单机游戏玩家设计的桌面应用程序，用于集中管理和下载游戏修改器（Trainer）。帮助用户快速获取和管理各类单机游戏辅助工具，告别繁琐的手动搜索和分散的资源管理。

---

## ✨ 功能特性

### 🎮 游戏浏览与发现

- 自动获取最新的游戏修改器列表
- 支持分页加载和滚动浏览
- 展示游戏封面图、名称等基本信息

### 🔍 游戏搜索

- 支持关键词搜索，实时显示结果
- 中英文关键词模糊匹配

### ⬇️ 下载管理

- 一键下载游戏修改器
- 实时显示下载进度、速度、剩余时间
- 自动处理文件名冲突
- 下载历史记录管理

### 📁 本地文件管理

- 自动扫描下载文件夹
- 支持 `.exe`、`.zip`、`.rar`、`.7z` 等格式
- 一键启动可执行文件
- 快速打开文件所在文件夹
- 智能匹配游戏封面图

### ⚙️ 设置管理

- 自定义下载文件夹路径
- 快速打开下载目录

## 🚀 快速开始

1. 下载最新版本的安装包或便携版
2. 运行安装程序或解压便携版
